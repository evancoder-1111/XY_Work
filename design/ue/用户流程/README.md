# 用户流程

本目录存放用户操作流程文档。

## 用户角色

1. **系统管理员**
   - 管理所有系统入口
   - 管理用户和权限
   - 查看系统统计

2. **普通用户**
   - 访问授权的系统入口
   - 查看个人待办任务
   - 查看通知消息

## 主要流程

### 登录流程

1. 用户访问系统
2. 选择登录方式（SSO 或用户名/密码）
3. 输入凭据
4. 系统验证
5. 登录成功，跳转工作台

### 工作台使用流程

1. 用户登录后进入工作台
2. 查看系统入口列表
3. 点击入口卡片
4. 系统通过 SSO 跳转到目标系统
5. 用户完成工作后返回工作台

### 入口管理流程（管理员）

1. 管理员进入入口管理页面
2. 查看入口列表
3. 执行操作（添加/编辑/删除）
4. 填写表单信息
5. 保存并刷新列表
