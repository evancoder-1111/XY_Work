# 交互设计

本目录存放交互流程图和交互规范文档。

## 交互流程

### 核心流程

1. **登录流程**
   - SSO 登录流程
   - 用户名/密码登录流程
   - 登录失败处理流程

2. **工作台操作流程**
   - 访问系统入口流程
   - 查看待办任务流程
   - 查看通知消息流程

3. **入口管理流程**（管理员）
   - 添加入口流程
   - 编辑入口流程
   - 删除入口流程

## 交互规范

### 反馈机制

- **操作成功**：顶部右侧 Toast 提示（3秒自动消失）
- **操作失败**：顶部右侧错误提示（可手动关闭）
- **加载状态**：骨架屏或加载指示器
- **确认操作**：重要操作使用模态对话框确认

### 交互细节

- 按钮点击有轻微动画反馈
- 卡片悬停有视觉变化
- 表单验证实时反馈
- 页面切换平滑过渡
