# XY Work 统一数字门户 - 信息架构

## 1. 整体架构

```
XY Work 统一数字门户
├── 公共区域
│   ├── 登录页
│   ├── 注册页（可选）
│   └── 忘记密码页
├── 工作台区域
│   ├── 个人工作台
│   │   ├── 系统入口
│   │   ├── 待办任务
│   │   ├── 通知消息
│   │   └── 个人统计
│   ├── 管理员工作台
│   │   ├── 系统入口管理
│   │   ├── 用户管理
│   │   ├── 角色管理
│   │   └── 统计报表
│   └── 部门工作台
│       ├── 部门成员管理
│       ├── 部门系统入口
│       └── 部门任务统计
└── 个人中心
    ├── 基本信息
    ├── 安全设置
    └── 系统偏好
```

## 2. 主要页面功能

### 2.1 登录页
- 用户名/密码登录
- SSO单点登录入口
- 忘记密码链接
- 记住密码选项

### 2.2 个人工作台
- 角色化系统入口展示
- 个性化收藏入口
- 待办任务列表
- 最新通知
- 快速搜索
- 个人使用统计

### 2.3 系统入口管理
- 入口列表展示
- 添加入口表单
- 编辑入口信息
- 删除入口确认
- 入口分类管理
- 入口权限设置

### 2.4 用户管理
- 用户列表展示
- 用户详情查看
- 添加/编辑用户
- 用户角色分配
- 用户状态管理

### 2.5 角色管理
- 角色列表展示
- 添加/编辑角色
- 权限分配
- 角色成员管理

## 3. 关键数据结构

### 3.1 用户数据
- 用户基本信息（ID、用户名、密码、姓名、邮箱、电话等）
- 用户角色关联
- 登录记录
- 操作日志

### 3.2 系统入口数据
- 入口基本信息（ID、名称、描述、图标、URL等）
- 访问权限设置
- 使用统计
- 创建/修改记录

### 3.3 角色数据
- 角色基本信息（ID、名称、描述等）
- 权限列表
- 关联用户

## 4. 权限层级

```
系统管理员 > 部门管理员 > 普通用户 > 访客
```

- **系统管理员**：完全访问权限，可管理所有功能
- **部门管理员**：管理部门内用户和入口
- **普通用户**：访问授权的系统入口和个人信息
- **访客**：有限的系统访问权限

## 5. 导航结构

### 5.1 顶部导航
- 系统标题/Logo
- 全局搜索
- 消息通知

### 5.2 侧边导航（管理员）
